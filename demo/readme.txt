로그인 관련 버그
1. 가입시, 모든 정보가 전부 기입되지 않으면 register 버튼이 활성화 되지 않게 조치하기
2. 회원가입 직후 ref,정보관리 등 모든 메뉴 클릭 시 오류 발생 그러나 로그아웃 후, 다시 접속하면 가능함 
- 500에러 발생 (HashMap cannot be cast to class com.example.demo.member.domain.SavedMemberDTO )
- 회원가입 시(/signup) 가입 완료 후, hashmap<String, SavedMemberDTO>에 userinfo라는 이름으로 로그인 정보를 담은 이후, 마찬가지로 userInfo라는 이름으로 세션에 저장
- 회원가입 직후에 에디트 메소드 및 getBoard 메소드는 실행됨
- session 값에서 userInfo 객체를 불러올 때, cast 변환 과정에서 문제가 있는 것으로 보임
- 로그인 시, hashmap에 로그인 정보를 담은 이후, 다시 세션에 그 값을 집어넣었는데, 불필요한 연산이며, 오류를 유발하기에 삭제하고, 저장 정보를 바로 세션에 담음으로써 정상 작동!
